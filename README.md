### How to install ?
``` bash
mkdir -p $HOME/.zshrc.d
cp mblet-prompt.zsh $HOME/.zshrc.d
```
``` bash
# add this in your $HOME/.zshrc
for file ($HOME/.zshrc.d/**/*.zsh) source $file
```
